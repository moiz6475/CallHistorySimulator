# Call History Simulator (Java DSA Project)

## Description
This project simulates a mobile call history system using Data Structures like Stack, Queue, Linked List, and Array.

## Features
- Add Contacts
- Search/Delete Contacts
- Missed Calls (Stack)
- Recent Calls (Queue)
- Favorite Contacts (Array)

## Technologies
- Java
- OOP Concepts
- Data Structures

## How to Run
1. Open project in IntelliJ
2. Run Main.java
3. Use menu system

## Data Structures Used
- Stack → Missed Calls
- Queue → Recent Calls
- Linked List → Contacts
- Array → Favorites
